How to configure Ubuntu to connect to OpenVPN

  1. Open Terminal
    In Unity
    Dash -> Search for Terminal
    In Gnome
    Application menu -> Accessories -> Terminal
    or
    Keyboard Shortcut: Ctrl + Alt + T
  2. Run the following command apt-get install network-manager-openvpn
  3. Right-click on the network icon on the system tray
  4. Choose: "Edit Connections"
  5. Click Add
  6. Choose Connection Type : openvpn
  7. The following window will open:
    1. Enter the connection name as example "VPN connection"
    2. Enter the Gateway
    3. Under the Authentication -> Type -> choose Passowrd
    4. Download our CA certificate
    5. Enter the username and the passowrd
    6. Import the CA certificate you just downloaded
  8. Click "Advanced" the following window will open:
    1. Set "Use Custom gateway port " to 999
    2. Mark "Use a TCP connection"
    3. Click "OK" and then Save
  9. Right-click on the network icon on the system tray
  10. Choose " VPN Connection then choose the new connection
